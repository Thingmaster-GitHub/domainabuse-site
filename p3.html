Routers and Scalability

DNS

The internet is a very, large, complicated system.Connecting all computers on the internet Directly would be expensive and impractical, so that is not how it works. Instead of doing this impossible task, it uses DNS. DNS stands for Domain Name System. this system allows URLS to be exchanged for the IP adress that leads to the server that you are trying to connect to. It has many redundant servers so that it if one server breaks it does not all go down with it, making the system fault tolerant. these many DNS servers are very scalable with different top level domans represented by different servers allowing for many different domains to be represented. these top level domains have a root domain located before them, allowing for even more variation. The number of these redundant servers make the system very reliable.

Wider internet

The wider internet aside from DNS also has simalar systems in place. There are many different routes, or paths that your connection can make when conecting to a server.The packets,small collections of information, are routed by a router to a server which sends back a response depending on the information sent. The rate of sending and recieving packets varies, this is called the bandwidth or bit rate. It is affected by the path that the information may take as well as the speed that your network is capable of handling. a simaler thing to this is called latency which is the amount of time that it takes to send and recieve packets

Conclusion

Essintially, all of these complicated systems are computer networks with many connected devices and a lot of redundant systems all working together to do various things like displaying sites like this one.
